# Decisiones de Conocimiento

Esta carpeta guarda decisiones de clasificacion cuando una fuente nueva cambia el sistema.

Ejemplos:

- una URL oficial obliga a crear un protocolo nuevo;
- una correccion de Rodrigo cambia una preferencia global;
- un documento externo actualiza una skill;
- una fuente deja obsoleto un workflow.

No guardar aqui notas sueltas. Cada decision debe incluir:

- fecha;
- fuente;
- cambio provocado;
- destino elegido;
- razon;
- archivos modificados;
- estado.
