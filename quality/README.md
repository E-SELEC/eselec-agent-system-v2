# Calidad de outputs

Esta carpeta contiene diagnosticos, criterios, contratos de salida y pruebas para mejorar la calidad real de los entregables E-SELEC.

Principio:

```text
La calidad no se arregla solo con prompts mas largos. Se arregla con contexto correcto, criterio explicito, verificacion y aprendizaje.
```

Usar esta carpeta para:

- diagnosticos de causas raiz;
- criterios por servicio;
- plantillas de evaluacion;
- contratos de output;
- pruebas de aprendizaje;
- decisiones de calidad del sistema.
