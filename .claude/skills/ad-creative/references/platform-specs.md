# Specs Ad Creative

## Limites rapidos

| Plataforma | Elemento | Limite operativo |
|---|---|---|
| Google RSA | headline | 30 caracteres |
| Google RSA | description | 90 caracteres |
| Google RSA | path | 15 caracteres |
| Meta | primary text | 125 visibles recomendado |
| Meta | headline | 40 recomendado |
| Meta | description | 30 recomendado |
| LinkedIn | intro text | 150 recomendado |
| LinkedIn | headline | 70 recomendado |
| TikTok | text | 80 recomendado |
| X/Twitter | tweet text | 280 caracteres |

## Angulos

- dolor;
- resultado;
- prueba/social proof;
- comparacion;
- objecion;
- urgencia real;
- identidad;
- localizacion;
- oferta directa;
- curiosidad sin clickbait.

## Control de calidad

- contar caracteres;
- eliminar duplicados;
- revisar claims;
- revisar CTA;
- revisar coherencia con landing;
- separar piezas por angulo;
- marcar piezas para test, no declarar ganadores sin datos.
