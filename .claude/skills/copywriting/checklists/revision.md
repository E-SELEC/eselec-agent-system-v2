# Checklist de revision - Copywriting

Antes de entregar, comprobar:

## Contexto

- [ ] Lei contexto, memoria, log, mensajes y tasks si existen.
- [ ] Revise brand/preferencias si aplica.
- [ ] Identifique oferta, audiencia y accion primaria.
- [ ] Declare fuentes y pruebas disponibles.

## Copy

- [ ] El headline es claro en 5 segundos.
- [ ] El subheadline explica como o por que.
- [ ] El CTA dice que pasa despues.
- [ ] Cada seccion tiene una idea.
- [ ] Beneficios conectan con problemas reales.
- [ ] El texto no podria servir igual para cualquier negocio.

## Riesgo

- [ ] No invente claims, cifras, testimonios, precios ni garantias.
- [ ] No use lenguaje legal/salud/finanzas sin fuente.
- [ ] No contradice contexto, log o brand.
- [ ] No se publica sin aprobacion si toca web, ads, email o CMS.

## Calidad

- [ ] Clasifique nivel CW0-CW3.
- [ ] Inclui variantes cuando aporta valor.
- [ ] Marque datos faltantes.
- [ ] Deje una siguiente accion unica.
