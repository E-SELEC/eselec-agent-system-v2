# Patrones Programmatic SEO

## Playbooks

- servicio + ciudad;
- categoria + ubicacion;
- integracion A + B;
- alternativa/comparativa;
- templates/ejemplos;
- directorio;
- glosario;
- calculadoras/conversiones;
- perfiles de entidades.

## Datos

Jerarquia:

1. propios;
2. producto/usuarios;
3. curados manualmente;
4. licenciados;
5. publicos enriquecidos.

## Riesgos

- thin content;
- doorway pages;
- canibalizacion;
- datos obsoletos;
- paginas huerfanas;
- indexacion masiva sin control;
- schema duplicado o falso.
