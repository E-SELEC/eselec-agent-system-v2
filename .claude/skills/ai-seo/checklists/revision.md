# Checklist de revision - AI SEO

Antes de entregar, comprobar:

## Evidencia

- [ ] Declare fecha, plataforma, query e idioma/mercado.
- [ ] Diferencie cita, mencion, enlace y trafico.
- [ ] No mezcle pruebas de dias distintos sin aclararlo.
- [ ] No use estadisticas sin fuente y fecha.
- [ ] Marque nivel AI0-AI3.

## SEO y contenido

- [ ] Revise si SEO tecnico bloquea la visibilidad.
- [ ] Revise schema cuando aplica.
- [ ] Revise arquitectura/hubs si aplica.
- [ ] Evalua extractabilidad: definicion, respuesta directa, tablas, FAQ, fuentes.
- [ ] Conecte la recomendacion a una pagina o query concreta.

## Crawlers y riesgo

- [ ] No recomende cambios de robots.txt sin verificar docs actuales.
- [ ] No toque robots.txt/CDN/WAF/noindex/sitemap sin Orden de Cambio.
- [ ] Diferencie bots de busqueda, entrenamiento y fetch de usuario.
- [ ] No prometa citas AI ni rankings.

## Cierre

- [ ] Deje una siguiente accion unica.
- [ ] Declare datos faltantes.
- [ ] No incluya secretos, PII, exports brutos ni capturas privadas.
