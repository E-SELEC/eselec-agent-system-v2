# Checklist - Popup CRO

- [ ] Lei contexto, memoria, log, mensajes y tasks si existen.
- [ ] Defini objetivo, oferta, audiencia y pagina/contexto.
- [ ] Revise trigger, frecuencia y exclusiones.
- [ ] Revise copy, CTA, cierre y declinar.
- [ ] Revise mobile, accesibilidad, consentimiento y SEO movil.
- [ ] Revise formulario interno si captura datos.
- [ ] Revise tracking de impression/close/submit/conversion.
- [ ] No propuse test sin baseline, trafico y tracking.
- [ ] No toque web/CMS/CMP/GTM/scripts sin Orden de Cambio.
- [ ] Clasifique PU0-PU3 y deje una siguiente accion unica.
