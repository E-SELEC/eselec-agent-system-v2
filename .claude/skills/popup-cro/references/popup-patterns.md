# Patrones Popup CRO

## Tipos

| Tipo | Uso | Riesgo |
|---|---|---|
| Click-triggered | lead magnet, demo, contenido gated | bajo si el usuario inicia |
| Scroll-based | blog, contenido largo, newsletter | interrupcion si aparece temprano |
| Exit intent | e-commerce, lead gen, objeciones finales | no existe igual en mobile |
| Sticky bar | anuncio o oferta simple | fatiga si queda permanente |
| Slide-in | CTA secundario o soporte | puede tapar contenido en mobile |
| Modal central | oferta fuerte o confirmacion | alto si bloquea contenido |

## Reglas de frecuencia

- maximo una interrupcion principal por sesion;
- recordar cierre o conversion;
- cooldown de dias, no minutos, para usuarios que cierran;
- excluir usuarios convertidos;
- evitar solapamiento con chat, cookie banner, newsletter y checkout.

## Mobile y SEO

- evitar interstitial grande antes del contenido;
- cierre visible y tactil;
- no cubrir navegacion critica;
- usar bottom sheet o banner si el objetivo es secundario;
- revisar que no bloquee contenido indexable.

## Medicion minima

- popup impression;
- close click;
- outside click / escape;
- form focus;
- submit attempt;
- conversion success;
- page bounce after popup;
- revenue/lead quality cuando aplique.
