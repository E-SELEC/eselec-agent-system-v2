# Checklist - Form CRO

- [ ] Lei contexto, memoria, log, mensajes y tasks si existen.
- [ ] Vi formulario, captura o lista de campos.
- [ ] Identifique objetivo y que pasa despues del envio.
- [ ] Justifique cada campo requerido.
- [ ] Revise mobile, labels, placeholders y errores.
- [ ] Revise privacidad y confianza cerca del formulario.
- [ ] Revise tracking de view/start/submit/success.
- [ ] No propuse test sin medicion/trafico.
- [ ] No toque formulario/CMS/CRM/GTM sin Orden de Cambio.
- [ ] Clasifique FC0-FC3 y deje una siguiente accion unica.
