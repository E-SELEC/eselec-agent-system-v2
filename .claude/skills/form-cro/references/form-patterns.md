# Patrones de formularios

## Contacto

Campos minimos:

- nombre;
- email o telefono;
- mensaje.

Mejoras:

- indicar tiempo de respuesta;
- alternativas: WhatsApp, telefono, email;
- privacidad breve.

## Demo / ventas

Campos frecuentes:

- nombre;
- email profesional;
- empresa;
- necesidad principal;
- telefono opcional;
- preferencia de contacto.

Evitar pedir presupuesto, equipo, cargo y mensaje largo si no se usa en ventas.

## Presupuesto / quote

Puede ser multi-step:

1. tipo de servicio;
2. detalles basicos;
3. contacto;
4. confirmacion.

Empezar con preguntas faciles. Dejar contacto sensible para el final.

## Lead magnet

Normalmente email basta.

Nombre puede ayudar a personalizar, pero reduce conversion. Pedir mas datos solo si el valor del recurso lo justifica.

## Checklist de campos

Para cada campo:

- que decision permite;
- quien lo usa;
- se puede obtener despues;
- puede ser opcional;
- genera friccion o desconfianza;
- requiere privacidad/consentimiento.
