# Checklist - Social Content

- [ ] Lei contexto, memoria, log, mensajes y marca disponible.
- [ ] Defini canal, objetivo, audiencia, tono y oferta.
- [ ] Cada pieza tiene objetivo y CTA.
- [ ] El copy se adapta al canal.
- [ ] No invente claims, cifras, testimonios ni promociones.
- [ ] Inclui recomendacion visual o asset requerido.
- [ ] Inclui medicion esperada.
- [ ] No publique ni programe sin Orden de Cambio.
- [ ] Clasifique SC0-SC3 y deje una siguiente accion unica.
