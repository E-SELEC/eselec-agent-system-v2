# Revision CRO de pagina

## 1. Propuesta de valor

- Se entiende que ofrece la pagina?
- Se entiende para quien es?
- Se entiende por que importa?
- Hay diferenciador real?
- El hero evita frases genericas?

## 2. Mensaje vs trafico

- La pagina coincide con el anuncio/query/email que trae al usuario?
- El titular responde a la intencion?
- Hay continuidad entre expectativa y contenido?

## 3. CTA y jerarquia

- Hay una accion primaria clara?
- El CTA esta visible arriba?
- El CTA dice que obtiene el usuario?
- Hay demasiadas acciones compitiendo?
- El CTA se repite en puntos logicos?

## 4. Confianza y prueba

- Hay logos, resenas, casos, datos o pruebas reales?
- La prueba esta cerca del claim?
- Hay datos especificos o solo frases vagas?
- La pagina responde "por que creerlo?"

## 5. Objeciones

- Precio/valor.
- Tiempo/esfuerzo.
- Confianza.
- Compatibilidad.
- Riesgo.
- "Esto es para mi?"

## 6. Friccion

- Formularios demasiado largos.
- Campos innecesarios.
- Navegacion distractora.
- Pasos siguientes confusos.
- Popups invasivos.
- Carga lenta o elementos pesados.

## 7. Mobile

- Hero visible sin cortar mensaje importante.
- CTA accesible.
- Formularios usables.
- Texto legible.
- Elementos no se solapan.
- Imagenes no empujan la accion demasiado abajo.

## 8. Accesibilidad basica

- Contraste suficiente.
- Botones identificables.
- Links descriptivos.
- Jerarquia de headings.
- Formularios con labels.

## 9. Medicion

- CTA trackeado.
- Form submit trackeado.
- Click WhatsApp/telefono trackeado.
- Conversion configurada.
- Fuente/campana identificable.

## 10. Prioridad

Prioriza por:

- impacto probable;
- gravedad del bloqueo;
- esfuerzo;
- dependencia;
- riesgo de implementacion.
