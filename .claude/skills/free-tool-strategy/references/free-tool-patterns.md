# Free Tool Patterns

## Tipos

| Tipo | Uso | Ejemplos |
|---|---|---|
| Calculator | decisiones numericas | ROI, ahorro, precio, coste |
| Generator | crear algo rapido | nombres, textos, templates |
| Analyzer | evaluar estado actual | SEO grader, web audit, score |
| Tester | comprobar si algo funciona | meta preview, speed check |
| Library | recurso reusable | snippets, icons, templates |
| Interactive | aprender o diagnosticar | quizzes, playgrounds, walkthroughs |

## Gating

- Ungated: maximo alcance, menor captura directa.
- Optional gated: buen equilibrio si el output basico se ve gratis.
- Partially gated: mostrar preview y pedir email para reporte completo.
- Fully gated: usar solo si el valor percibido es muy alto.

## Scorecard 1-5

- Search demand.
- Audience match.
- Natural path to product.
- Build feasibility.
- Maintenance burden inverso.
- Link potential.
- Share-worthiness.
- Differentiation vs existing tools.

25+ fuerte, 15-24 prometedor, menos de 15 reconsiderar.

## Riesgos

- Calculos opacos o falsos.
- Herramienta demasiado amplia.
- Lead magnet disfrazado sin utilidad real.
- Captura excesiva.
- Coste de mantenimiento invisible.
- No medir activacion ni conversion posterior.
