<div className="digest-wins">
  <p className="digest-wins-title">Otros logros</p>

  <div className="digest-wins-grid">
    <div><a href="/docs/es/interactive-mode#vim-editor-mode">Modo visual de Vim</a>: presione <code>v</code> para selección de caracteres o <code>V</code> para selección de líneas en la entrada del prompt, con operadores y retroalimentación visual</div>
    <div>Los hooks ahora pueden llamar directamente a herramientas MCP a través de <a href="/docs/es/hooks#mcp-tool-hook-fields"><code>type: "mcp\_tool"</code></a>, por lo que un hook puede acceder a un servidor ya conectado sin generar un proceso</div>
    <div><code>/cost</code> y <code>/stats</code> se fusionan en <a href="/docs/es/commands"><code>/usage</code></a>; los nombres antiguos aún funcionan como atajos de escritura que abren la pestaña relevante</div>
    <div>Los cambios de <code>/config</code> (tema, modo de editor, verbose y similares) ahora persisten en <code>\~/.claude/settings.json</code> y siguen la misma precedencia de proyecto/local/política que otras <a href="/docs/es/settings">configuraciones</a></div>
    <div>Los <a href="/docs/es/sub-agents#fork-the-current-conversation">subagentes bifurcados</a> se pueden habilitar en compilaciones externas con <code>CLAUDE\_CODE\_FORK\_SUBAGENT=1</code>: una bifurcación hereda su contexto de conversación completo en lugar de comenzar de nuevo</div>
    <div>El <a href="/docs/es/model-config#adjust-effort-level">nivel de esfuerzo</a> predeterminado para suscriptores Pro y Max en Opus 4.6 y Sonnet 4.6 es ahora <code>high</code> (era <code>medium</code>)</div>
    <div>Las compilaciones nativas de macOS y Linux reemplazan las herramientas <code>Glob</code> y <code>Grep</code> con <code>bfs</code> y <code>ugrep</code> integrados disponibles a través de Bash, para búsquedas más rápidas sin un viaje de herramienta separado</div>
    <div><code>--from-pr</code> ahora acepta URLs de solicitud de fusión de GitLab, solicitud de extracción de Bitbucket y PR de GitHub Enterprise además de github.com</div>
    <div>Modo automático: incluya <code>"\$defaults"</code> en <a href="/docs/es/auto-mode-config"><code>autoMode.allow</code>, <code>soft\_deny</code>, o <code>environment</code></a> para agregar reglas personalizadas junto con la lista integrada en lugar de reemplazarla</div>
    <div>El nuevo comando <a href="/docs/es/plugin-dependencies#tag-plugin-releases-for-version-resolution"><code>claude plugin tag</code></a> crea etiquetas git de lanzamiento para plugins con validación de versión</div>
    <div>Las sesiones de Opus 4.7 ahora se calculan contra la ventana de contexto nativa de 1M del modelo, corrigiendo porcentajes inflados de <code>/context</code> y autocompactación prematura</div>
    <div><code>/resume</code> en sesiones grandes es hasta un 67% más rápido y ahora ofrece resumir sesiones grandes y obsoletas antes de releerlas</div>
  </div>
