# Checklist - A/B Test Setup

- [ ] Lei contexto, memoria, log, mensajes y tasks si existen.
- [ ] La hipotesis parte de una observacion o dato.
- [ ] Hay un cambio principal por test.
- [ ] Hay metrica primaria clara.
- [ ] Hay metricas secundarias y guardrails.
- [ ] Revise baseline, trafico, MDE y muestra.
- [ ] No propuse test si falta tracking.
- [ ] No llame ganador a resultado inconcluso.
- [ ] No toque produccion sin Orden de Cambio.
- [ ] Clasifique AB0-AB3 y deje siguiente accion unica.
