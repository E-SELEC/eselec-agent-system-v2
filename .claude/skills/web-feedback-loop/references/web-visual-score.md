# Web Visual Score

## Dimensiones

| Dimension | Peso | Evalua |
|---|---|---|
| Estructura | 20 | flujo, secciones, orden, claridad |
| Jerarquia | 20 | H1, H2, tamanos, contraste visual |
| Legibilidad | 20 | fuente, contraste, line length, mobile |
| CTA | 20 | visibilidad, copy, contraste, repeticion |
| Marca | 20 | colores, tono, imagenes, coherencia |

## Umbrales

- 80-100: listo para presentar.
- 70-79: bueno, pulir detalles.
- 50-69: necesita iteracion.
- 0-49: replantear estructura visual.

## Brechas comunes

- Hero sin CTA claro.
- H1 compite con subtitulos.
- Texto pequeno en mobile.
- Secciones sin respiracion.
- Paleta inconsistente.
- Imagenes stock o irrelevantes.
- CTA generico.

## Recomendacion

Una iteracion debe tocar pocas cosas y medir antes/despues. Si hay muchos problemas, separar por rondas.
