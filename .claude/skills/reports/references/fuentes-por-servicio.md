# Fuentes por servicio - Reports

Usar esta referencia para decidir que datos buscar antes de redactar informes, alertas o proximos pasos.

## Regla base

No todos los informes necesitan todas las fuentes. Solo exigir la fuente que puede cambiar la decision del output.

Si una fuente falta:

- declarar que falta;
- explicar que conclusion queda limitada;
- no inventar rendimiento;
- decidir si el output es R0, R1, R2 o R3 segun `SKILL.md`.

## SEO organico

Fuentes principales:

- GSC: clics, impresiones, CTR, queries, paginas, fechas.
- SEMrush: visibilidad estimada, keywords, competidores, gaps, backlinks.
- Log del cliente: acciones SEO ejecutadas.
- Outputs SEO: auditorias, planes, cambios de contenido, arquitectura.

Lectura esperada:

- que subio, bajo o quedo igual;
- que paginas o queries explican el cambio;
- que accion SEO se hizo y si ya puede medirse;
- que falta para afirmar causalidad.

## SEO local

Fuentes principales:

- GBP: llamadas, visitas, interacciones, reseñas, direccion, fotos.
- GSC: consultas locales y paginas de servicio local.
- SEMrush: competidores locales y directorios.
- `mensajes.md`: alertas de reputacion o perfil.

Lectura esperada:

- visibilidad local;
- reputacion;
- acciones locales hechas;
- proxima mejora de ficha, reseñas o paginas locales.

## Paid media / SEM

Fuentes principales:

- plataforma Ads: gasto, impresiones, clics, leads, CPA/ROAS cuando exista.
- GA4/GTM: conversiones y eventos.
- Landing page: coherencia oferta-anuncio.
- Log del cliente: cambios de campaña ejecutados.

Lectura esperada:

- si el presupuesto compro aprendizaje, leads o ventas;
- si el problema esta en anuncios, audiencia, tracking o landing;
- si hay datos suficientes para escalar, pausar o solo observar.

## CRO

Fuentes principales:

- GA4: eventos, conversiones, funnel, paginas.
- Capturas o URL: propuesta de valor, CTA, friccion.
- Outputs CRO: auditorias, tests, cambios sugeridos.
- Mensajes: bloqueos de web, formulario o tracking.

Lectura esperada:

- que friccion afecta conversion;
- que dato soporta la hipotesis;
- que cambio conviene hacer o testear.

## Social

Fuentes principales:

- plataforma social o export: alcance, interacciones, seguidores, contenido top.
- Calendario/publicaciones ejecutadas.
- Contexto de marca y tono.
- Objetivo del canal.

Lectura esperada:

- que contenido genero respuesta;
- que aprendizaje aplica al proximo periodo;
- que no se puede afirmar solo con likes o seguidores.

## Web / WooCommerce

Fuentes principales:

- estado de web/CMS si fue revisado.
- GA4/GTM: trafico, conversiones, eventos clave.
- WordPress/WooCommerce: solo si hay acceso seguro y permiso.
- Outputs web: auditorias, cambios, QA, planes de lanzamiento.

Lectura esperada:

- que cambio afecta SEO, conversion o operacion;
- si hay riesgos tecnicos;
- si requiere Orden de Cambio antes de tocar produccion.

## Agencia / comunicacion interna

Fuentes principales:

- `agency/context.md`
- `agency/log.md`
- `agency/mensajes.md`
- `agency/brand.md`
- preferencias de Rodrigo.

Lectura esperada:

- decision recomendada;
- riesgos;
- siguiente accion;
- que queda fuera.
