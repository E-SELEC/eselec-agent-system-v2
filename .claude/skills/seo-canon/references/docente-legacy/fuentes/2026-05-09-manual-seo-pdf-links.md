## Aprendizaje del Docente SEO

El marco de Rodrigo queda reforzado con tres tipos de fuentes:

1. **Google:** autoridad oficial para requisitos, documentacion, medicion y politicas.
2. **Semrush:** ejecucion con herramientas, diagnostico, investigacion y seguimiento.
3. **Ahrefs:** apoyo pedagogico para keyword research, on-page y tecnico.

Como Docente SEO, debo diferenciar:

```text
Fuente oficial = define reglas o requisitos.
Herramienta SEO = ayuda a investigar, diagnosticar y priorizar.
Guia sectorial = ayuda a explicar y formar, pero debe contrastarse.
```

## Pendiente

Antes de construir una prueba o modulo oficial para agentes reales:

1. verificar enlaces clave;
2. separar fuentes obligatorias de fuentes de apoyo;
3. decidir que fuentes son necesarias para web nueva;
4. decidir que fuentes son necesarias para web existente;
5. convertir enlaces en ruta de estudio por niveles.
