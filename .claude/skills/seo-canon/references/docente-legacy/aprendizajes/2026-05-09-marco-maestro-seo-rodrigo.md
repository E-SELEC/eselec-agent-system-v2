## 2. Diferencia clave: web nueva vs web existente

Esta es una distincion central del marco.

```text
WEB NUEVA
Se disena el sistema SEO antes de construir.
Primero estrategia.
Luego arquitectura.
Luego contenido.
Luego desarrollo.
Luego publicacion.
Luego medicion.
```

```text
WEB EXISTENTE
Se audita lo que ya esta vivo antes de tocar.
Primero medir.
Luego proteger.
Luego corregir.
Luego optimizar.
Luego expandir.
Luego medir.
```

Reglas profesionales:

```text
En una web nueva:
No construyas sin estrategia SEO.

En una web existente:
No cambies nada importante sin saber que valor tiene.
```

## 7. Canibalizacion

La canibalizacion ocurre cuando dos o mas URLs compiten por la misma intencion
de busqueda.

```text
CANIBALIZACION SEO
|
|-- Que es
|   |-- Dos o mas URLs compiten por la misma intencion de busqueda.
|
|-- Cuando ocurre
|   |-- Mismo tema
|   |-- Mismo keyword objetivo
|   |-- Mismo tipo de contenido
|   |-- Mismo usuario objetivo
|   |-- Mismo embudo
|   |-- Misma promesa
|
|-- Como se corrige
|   |-- Fusionar
|   |-- Redirigir
|   |-- Diferenciar intencion
|   |-- Cambiar enfoque
|   |-- Cambiar keyword principal
|   |-- Ajustar enlaces internos
|   |-- Reforzar la URL principal
|
|-- Regla
    |-- Una URL principal por intencion principal.
```

No todas las paginas parecidas canibalizan. Puede haber varias URLs sobre el
mismo tema si cada una cumple una intencion distinta.

Ejemplo que no canibaliza:

```text
/seo-local-madrid/              -> landing comercial
/blog/que-es-seo-local/         -> articulo informacional
/blog/seo-local-para-restaurantes/ -> guia sectorial
/casos/seo-local-clinica-madrid/ -> caso de exito
```

Arquitectura SEO no es solo menu bonito. Es como se distribuyen intencion,

```text
|   |-- Servicios clave

|   |-- Productos clave
|   |-- Landings locales
|   |-- Paginas comerciales
|
|-- Paginas secundarias
|   |-- Subservicios
|   |-- Subcategorias
|   |-- Variantes
|   |-- Segmentos
|   |-- Zonas
|
|-- Contenido de apoyo
|   |-- Blog
|   |-- Guias
|   |-- Comparativas
|   |-- FAQs
```

```text
accesos utiles como sitelinks.
```

```text
|   |-- Expansion controlada

    |-- SEO de contenidos
    |-- SEO local
    |-- SEO Off-Page

    |-- CRO

```

Frase resumen:

```text
En una web nueva, el SEO disena el sistema antes de construirlo.
En una web existente, el SEO diagnostica, protege y mejora antes de expandir.
```

Regla de oro:

```text
Una URL = una intencion principal.
Una intencion importante = una URL principal.
Todo lo demas debe apoyar, no competir.
```
