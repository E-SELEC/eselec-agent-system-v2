# Checklist - Competitor Alternatives

- [ ] Defini formato y query objetivo.
- [ ] Investigue competidor con fuentes fechadas.
- [ ] Verifique pricing/features/claims.
- [ ] Reconoci fortalezas del competidor y limites propios.
- [ ] Inclui para quien conviene cada opcion.
- [ ] Defini URL, metadata, estructura, enlaces y schema/FAQ si aplica.
- [ ] No difame ni invente claims.
- [ ] No propuse publicar sin Orden de Cambio.
- [ ] Clasifique CA0-CA3 y deje una siguiente accion unica.
