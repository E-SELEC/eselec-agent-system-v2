# Patrones Competitor Alternatives

## Formatos

- `[Competitor] alternative`: usuario quiere cambiar.
- `[Competitor] alternatives`: usuario compara opciones.
- `[You] vs [Competitor]`: comparacion directa.
- `[A] vs [B]`: comparacion de terceros con opcion propia.

## Secciones utiles

- TL;DR;
- tabla comparativa;
- criterios de decision;
- fortalezas de cada opcion;
- limites de cada opcion;
- pricing fechado;
- para quien es cada herramienta;
- migracion/cambio;
- FAQ;
- CTA.

## Fuentes

- sitio oficial;
- pricing oficial;
- docs/changelog;
- reviews verificables;
- experiencia propia;
- clientes que cambiaron;
- SERP/competidores con fecha.
