# plan-lead-magnet

Planifica un lead magnet sin publicarlo ni capturar leads.

## Uso

```text
/plan-lead-magnet [agencia/cliente] [audiencia/problema] [--write]
```

## Workflow

1. Leer `.claude/skills/lead-magnets/SKILL.md`.
2. Leer contexto de agencia o cliente y `quality/criterios-output.md`.
3. Confirmar audiencia, problema, etapa y siguiente paso.
4. Entregar usando `templates/lead-magnet-plan.md`.

## Reglas

- No publicar landing.
- No crear formularios reales.
- No activar capturas ni automatizaciones.
- Si hay implementacion real, abrir Orden de Cambio.
