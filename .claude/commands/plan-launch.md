# plan-launch

Planifica un lanzamiento, beta, feature release o Product Hunt sin publicarlo.

## Uso

```text
/plan-launch [agencia/cliente] [producto/feature] [--write]
```

## Workflow

1. Leer `.claude/skills/launch-strategy/SKILL.md`.
2. Leer contexto de agencia o cliente y `quality/criterios-output.md`.
3. Confirmar etapa, audiencia, objetivo, fecha y canales.
4. Entregar usando `templates/launch-plan.md`.

## Reglas

- No publicar anuncios.
- No enviar emails.
- No abrir accesos ni waitlists reales.
- No tocar web, producto o cuentas externas sin Orden de Cambio.
