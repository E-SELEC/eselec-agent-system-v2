# aplicar-psicologia-marketing

Revisa un activo, mensaje o flujo con psicologia de marketing etica.

## Uso

```text
/aplicar-psicologia-marketing [agencia/cliente] [activo/flujo] [--write]
```

## Workflow

1. Leer `.claude/skills/marketing-psychology/SKILL.md`.
2. Leer contexto de agencia o cliente y `quality/criterios-output.md`.
3. Confirmar comportamiento objetivo, etapa, friccion y evidencia.
4. Entregar usando `templates/psychology-review.md`.

## Reglas

- No proponer dark patterns.
- No implementar cambios reales.
- Cambios en web, pricing, onboarding o emails requieren Orden de Cambio.
