# plan-free-tool

Planifica una herramienta gratuita de marketing sin construirla ni publicarla.

## Uso

```text
/plan-free-tool [agencia/cliente] [idea/problema] [--write]
```

## Workflow

1. Leer `.claude/skills/free-tool-strategy/SKILL.md`.
2. Leer contexto de agencia o cliente y `quality/criterios-output.md`.
3. Confirmar audiencia, problema, objetivo, recursos y oferta relacionada.
4. Entregar usando `templates/free-tool-plan.md`.

## Reglas

- No crear formularios reales.
- No activar tracking real.
- No publicar herramienta o landing.
- Implementacion requiere Orden de Cambio.
