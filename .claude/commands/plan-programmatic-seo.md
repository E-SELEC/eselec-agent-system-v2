# plan-programmatic-seo

Planifica paginas SEO a escala sin publicarlas.

## Uso

```text
/plan-programmatic-seo [cliente] [patron] [--write]
```

## Workflow

1. Leer `.claude/skills/programmatic-seo/SKILL.md`.
2. Leer contexto, memoria, log, mensajes, tasks, manifest y `quality/criterios-output.md`.
3. Confirmar patron, fuente de datos, template, arquitectura e indexacion.
4. Entregar usando `templates/programmatic-seo-plan.md`.

## Reglas

- No publicar paginas.
- No tocar URLs, sitemap, CMS ni noindex.
- No crear thin content.
- Si hay implementacion real, abrir Orden de Cambio.
