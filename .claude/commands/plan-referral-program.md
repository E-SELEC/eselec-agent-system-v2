# plan-referral-program

Disena un programa de referidos o afiliados sin lanzarlo.

## Uso

```text
/plan-referral-program [agencia/cliente] [tipo/objetivo] [--write]
```

## Workflow

1. Leer `.claude/skills/referral-program/SKILL.md`.
2. Leer contexto y `quality/criterios-output.md`.
3. Confirmar tipo, objetivo, incentivo, tracking y restricciones.
4. Entregar usando `templates/referral-program-plan.md`.

## Reglas

- No lanzar programa.
- No configurar pagos, links ni herramientas.
- No enviar emails.
- Si hay implementacion real, abrir Orden de Cambio.
