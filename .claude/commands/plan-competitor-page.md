# plan-competitor-page

Planifica una pagina de alternativas o comparacion sin publicarla.

## Uso

```text
/plan-competitor-page [cliente] [competidor/formato] [--write]
```

## Workflow

1. Leer `.claude/skills/competitor-alternatives/SKILL.md`.
2. Leer contexto, memoria, log, mensajes, tasks, manifest y `quality/criterios-output.md`.
3. Confirmar formato, competidor, query, fuentes y posicionamiento.
4. Entregar usando `templates/competitor-page-plan.md`.

## Reglas

- No publicar paginas.
- No inventar features, pricing ni reviews.
- No difamar competidores.
- Si hay implementacion real, abrir Orden de Cambio.
