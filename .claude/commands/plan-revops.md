# plan-revops

Disena proceso RevOps sin tocar CRM real.

## Uso

```text
/plan-revops [agencia/cliente] [problema] [--write]
```

## Workflow

1. Leer `.claude/skills/revops/SKILL.md`.
2. Leer contexto y `quality/criterios-output.md`.
3. Confirmar motion, stack, fuente de verdad, objetivo y problema.
4. Entregar usando `templates/revops-spec.md`.

## Reglas

- No modificar CRM.
- No importar datos.
- No activar workflows.
- Si hay implementacion real, abrir Orden de Cambio.
