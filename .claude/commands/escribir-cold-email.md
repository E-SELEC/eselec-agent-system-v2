# escribir-cold-email

Escribe emails frios o follow-ups sin enviarlos.

## Uso

```text
/escribir-cold-email [agencia/cliente] [audiencia/oferta] [--write]
```

## Workflow

1. Leer `.claude/skills/cold-email/SKILL.md`.
2. Leer contexto de agencia o cliente y `quality/criterios-output.md`.
3. Confirmar audiencia, senal, oferta, prueba y CTA.
4. Entregar usando `templates/cold-email-sequence.md`.

## Reglas

- No enviar emails.
- No automatizar secuencias.
- No usar datos personales no autorizados.
- Si hay envio real, abrir Orden de Cambio.
