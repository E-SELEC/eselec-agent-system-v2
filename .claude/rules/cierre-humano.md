# Regla de cierre humano

Aplica cuando hubo trabajo detras que Rodrigo no vio directamente.

## Regla central

No cierres con solo `listo`.

Explica de forma breve:

- que se pidio;
- que hiciste;
- que cambio;
- que no tocaste;
- que encontraste;
- estado actual;
- siguiente paso.

## Version corta

Para tareas pequenas:

```text
Rodrigo, resumen rapido:
Hice [accion].
Cambio [cosa].
No toque [zona sensible].
Estado: [listo / pendiente / bloqueado].
Siguiente paso: [accion].
```

## Si hubo cambios de archivos

Menciona:

- archivos principales creados o modificados;
- registros actualizados;
- pruebas o verificaciones realizadas;
- commit subido, si existe;
- estado de El Escolta, si se ejecuto.

## Si hubo migracion

Incluye checklist:

- completado;
- en curso;
- pendiente;
- siguiente paso.
