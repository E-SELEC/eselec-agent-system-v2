# Memory - Computer Chamberi

## Que ha funcionado

- SEO local avanzado ya aplicado y funcionando como base del proyecto.
- Dominio de busquedas de marca.
- GBP fuerte como activo principal de captacion local.
- Queries en ingles muestran oportunidad real para publico internacional en Madrid.
- Nichos Amazfit, Xiaomi y GoPro ya generan demanda organica.

## Que no ha funcionado

- CTR organico bajo: hay impresiones y posiciones, pero muchos resultados no capturan clics suficientes.
- Calidad de backlinks baja segun diagnostico legacy.
- Instagram no aporta valor actual.
- UX/UI incompleta puede limitar conversion.
- GA4/conversiones deben verificarse antes de informes de negocio.

## Preferencias y restricciones

- No invertir esfuerzo en Instagram salvo decision explicita.
- Mantener enfoque bilingue/ingles cuando tenga sentido para el mercado internacional.
- No hacer cambios tecnicos sin confirmar CMS, acceso y riesgo.
- GSC/GBP se tratan como accesos sensibles: usar solo via canal aprobado y sin guardar credenciales.

## Hipotesis a probar

- Optimizar titles/metas en paginas con muchas impresiones y bajo CTR puede aumentar clics sin crear trafico nuevo.
- Paginas dedicadas a Amazfit/Xiaomi/GoPro pueden capturar trafico de alta intencion.
- CRO en paginas clave puede mejorar leads cuando la medicion este verificada.
- Link building editorial/local puede mejorar autoridad si se hace con calidad.
