# Sistema de prioridad

## Urgente

Puede afectar produccion, gasto, reputacion, ventas, acceso, datos o experiencia de cliente de forma inmediata.

## Importante

Impacta resultados, calidad o velocidad, pero no rompe una operacion viva.

## Rutinario

Mantenimiento, documentacion, mejoras no bloqueantes o preparacion.

El lider clasifica antes de ejecutar y explica la decision si hay riesgo.
